# My-Data-Portfilio

## Dataset Information

This dataset contains information on:

-Overall attendance
-Authorised and unathorised absences
-Illness absences
-Lateness
-Educational activity
-School enrolments
-Regional attendance statistics

Key columns include:

- region_name
- education_phase
- la_name
- attendance_date
- attendance_perc
- overall_absence_perc
